# 附加页交互整理

日期：2026-06-20

## 目标
- 附加页可独立选择 SIM，不再要求用户回到 IMS 页切换。
- APN 模块按真实能力表达：系统 APN 设置入口 + 推荐 APN 写入。
- 快捷开关模块提供实际添加入口，不只显示说明文字。
- 配置备份明确为“当前 SIM 配置备份”，避免误解为备份所有 SIM 或 IMS 注册状态。
- 附加页按使用顺序重排，先确定 SIM，再做区域、APN、网络、抖音、快捷开关和备份。

## 非目标
- 不新增新的运营商 APN 规则库。
- 不自动判断 IMS APN 是否一定正确。
- 不修改 IMS 首页核心开关逻辑。
- 不备份 IMS 注册状态。

## 行为变化
- 附加页顶部显示当前 SIM，可直接切换 SIM。
- 附加页从“所有 SIM”状态进入时默认使用第一张真实 SIM；APN 写入、TikTok 修复、备份与恢复都使用附加页当前可见 SIM。
- 区域兼容和 APN 卡片优先显示系统返回的两位 ISO；如果系统返回值不是两位字母，则按 MCC 推导常见国家码，例如 460 显示 cn。
- APN 写入按钮改为“写入推荐 APN”，确认弹窗展示将写入的名称、APN 和类型。
- 打开 APN 设置前要求选中单张 SIM，并把当前 SIM 的 subId 传给系统设置；失败时给出明确提示。
- 支持 Android 13+ 通过系统能力请求添加 QS 图块；不支持时保留手动说明。
- 配置备份保存当前 SIM 的功能开关快照、MCC/MNC/ISO 和国家码覆盖，并在界面上明确说明。

## 影响范围
- 附加页 UI 和操作顺序。
- APN 设置入口与推荐 APN 写入入口。
- 快捷设置图块添加入口。
- 配置备份文案和列表展示。

## 验证方式
- 运行单元测试和 Release 构建。
- 手动验证附加页可直接切 SIM。
- 手动验证 APN 设置入口能打开当前 SIM 的 APN 页面，失败时有提示，写入推荐 APN 前有确认。
- Android 13+ 设备验证快捷开关添加请求可弹出系统确认。
